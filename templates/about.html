About This Project

This web application is part of the Banking and Flight Service Churn Prediction Project.

It uses a Random Forest Machine Learning Model trained on customer financial and service-related data to predict the likelihood of churn.

Features used in the prediction include credit score, age, tenure, balance, activity, products, and salary.

The system helps financial institutions and airlines take proactive measures to retain valuable customers.
